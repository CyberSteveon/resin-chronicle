# Resin Chronicle

> 35 dark IDE color themes spanning Neon Resin, Aztec Resin, Fantasy, and Final Fantasy worlds.

## Theme Packs

| Pack | Themes | Vibe |
|------|--------|------|
| Neon Resin | 10 | Cyberpunk / synthwave resin |
| Aztec Resin | 2 | Deep navy and void |
| Fantasy | 7 | High fantasy worlds |
| Final Fantasy | 16 | FF7 through FF16 |

## Installation

### Sublime Text
Copy all `.sublime-color-scheme` files from `sublime/` into:
```
~/Library/Application Support/Sublime Text/Packages/User/
```

### JetBrains (IntelliJ, PyCharm, RustRover, etc.)
Copy all `.icls` files from `jetbrains/` into:
```
~/Library/Application Support/JetBrains/<IDE><version>/colors/
```

### VS Code
Install via VSIX (see Releases) or load in development mode:
```bash
code --extensionDevelopmentPath=/Users/steven/Projects/resin-chronicle/vscode
```
